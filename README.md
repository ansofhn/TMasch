# T-Masch (Ticketing Management System Administration School)

T-Masch is a desktop-based administrative ticketing management system designed to streamline school operational workflows. Developed as a final project for the **Visual Programming** course, this application allows administrators to efficiently manage users, roles, and incoming tickets.

## 🚀 Features
* **User Management:** Full CRUD (Create, Read, Update, Delete) operations for school staff and users.
* **Role-Based Access Control:** Efficiently categorizes users into Admin, Guru, and Siswa.
* **Interactive Dashboard:** Real-time data synchronization between the database and the UI using Java Swing components.
* **Optimized Data Flow:** Uses Integer-based mapping for roles to ensure database performance and consistency.

## 🛠 Tech Stack
* **Language:** Java
* **IDE:** Apache NetBeans
* **UI Framework:** Swing (JFrame)
